# 포인트 제도 정책

## 1. 포인트 적립 (Point Accumulation)
-   **구매 적립**: 상품 구매 확정 시 결제 실 결제 금액의 **1%가 기본 적립**됩니다. (멤버십 등급에 따라 최대 5%까지 차등 적립)
-   **리뷰 적립**: 텍스트 리뷰 작성 시 **0.36포인트**, 포토 리뷰 작성 시 **0.71포인트**를 지급합니다.

## 2. 포인트 사용 (Point Usage)
-   포인트는 **1포인트당 1달러(USD)**의 가치를 가집니다.
-   최소 **0.71포인트 이상** 보유 시 결제 단계에서 현금처럼 사용 가능합니다.

## 3. 포인트 소멸 (Point Expiration)
-   적립된 포인트의 유효기간은 적립일로부터 **1년**입니다.
-   유효기간 경과 시 선입선출(FIFO) 방식에 의해 자동 소멸됩니다.

## 4. 반품 및 취소 시 처리 (Point Rollback)
-   **적립 포인트 회수**: 상품 반품 시 해당 구매로 적립되었던 포인트는 자동으로 회수됩니다.
-   **사용 포인트 환불**: 포인트로 결제한 금액은 반품 시 포인트로 우선 환불됩니다.
-   **조건 미달 시 차감**: 포인트 사용 후 주문 일부를 취소하여 남은 결제 금액이 포인트 사용 기준(최소 주문금액 등)에 미달이 될 경우, 사용된 포인트는 환급되며 실 결제 수단에서 해당 금액이 차감될 수 있습니다.
